🔧 Encanadores Mario & Luigi

Landing page desenvolvida para simular um serviço de orçamento e atendimento hidráulico, com foco em conversão de clientes e experiência do usuário.

📌 Sobre o projeto

Este projeto consiste em uma página web inspirada no universo do Mario Bros, criada para representar uma empresa fictícia de encanadores.

A proposta é oferecer ao usuário uma forma rápida e intuitiva de entrar em contato e solicitar um orçamento para serviços hidráulicos.

🚀 Funcionalidades

✔️ Navegação simples e objetiva
✔️ Seção de apresentação do serviço
✔️ Botão de chamada para ação (Entre em contato)
✔️ Integração com WhatsApp
✔️ Layout responsivo
✔️ Design atrativo com identidade visual temática

🎯 Objetivo do projeto
Simular uma landing page real de prestação de serviços
Praticar construção de páginas com foco em conversão
Trabalhar responsividade e organização de layout
Aplicar conceitos de UI/UX

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript

🎨 Diferenciais

✨ Design criativo inspirado no Mario Bros
✨ Foco em conversão (call to action bem destacado)
✨ Integração com WhatsApp para contato rápido
✨ Estrutura pensada como projeto real de cliente
